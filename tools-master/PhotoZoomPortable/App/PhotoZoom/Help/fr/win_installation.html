Installation pour Windows

Pour installer PhotoZoom Pro 5, lancez le fichier Setup ou placez le CD dans un lecteur. Si la fonction "Auto-exécution" de vos lecteurs n'est pas active, consultez le contenu du CD et cliquez sur le programme Setup.

Après avoir lu la license d'utilisation, cochez la case "J'accepte les conditions" pour faire apparaître le bouton "Suivant" :



Pendant l'installation, vous pouvez personnaliser l'emplacement et la manière dont PhotoZoom Pro 5 sera installé sur votre ordinateur. Il n'est cependant pas nécessaire de modifier les paramètres par défaut de l'installateur. Il vous suffit de suivre l'affichage des fenêtres en cliquant sur "Installer" et éventuellement sur "Terminer".
Lorsque l'installation sera terminée, vous pourrez lancer PhotoZoom Pro 5 en double cliquant sur l'icône placée sur votre bureau, ou dans me menu "Démarrer":

Les modules complémentaires "Automatisation" et "Exportation" d'Adobe ® Photoshop ®

PhotoZoom Pro 5 peut être utilisé comme une application autonome ("stand-alone") ou en conjonction avec Adobe Photoshop, comme module complémentaire des fonctions "Automatisation" et "Exportation". Ces dernières fonctions ne seront installées que si Photoshop fonctionne déjà sur votre ordinateur avant l'installation de PhotoZoom Pro 5. Le processus d'installation détecte automatiquement où installer les modules complémentaires. Vous pouvez également spécifier un dossier particulier, mais dans ce cas, vérifiez que Photoshop est paramétré pour rechercher les filtres dans ce dossier ( Edition > Préférences > Modules externes dans Photoshop).

Après avoir installé le module complémentaire, PhotoZoom Pro 5 apparaîtra lorsque vous cliquerez sur Fichier > Automatisation ou sur Fichier > Exportation. Vous devrez redémarrer Photoshop pour que le module complémentaire soit reconnu.

Autres programmes hôtes

Dans Adobe® Photoshop® Elements, le module complémentaire (plug-in) Automatisation est accessible soit par la commande "Fichier" > "Outils d'automatisation", soit par "Fichier" > "Automatisation", en fonction de votre version. Le module d'Exportatilon est accessible par "Fichier" > "Exportation".

Dans Corel® PHOTO PAINT, le module complémentaire (plug-in) est accessible par la commande "Fichier" > "Plug-ins de format" > "Export".

Dans Corel® PaintShop Pro (Photo), le module complémentaire (plug-in) d'exportation est accessible par la commande "Fichier" > "Export" > "Plugins".


PhotoZoom Pro 5 peut être utilisé comme module complémentaire (plug-in) avec de nombreux logiciels. Consultez Resources système pour une description complète.
