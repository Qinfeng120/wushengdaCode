Menu e barra degli strumenti

La barra degli strumenti contiene i seguenti pulsanti:

Apri
Consente di aprire un'immagine esistente nella finestra di anteprima.

Salva
Consente di salvare le immagini ridimensionate dopo aver scelto una posizione e specificato nome del file e formato. Sono supportati i formati comuni come TIFF, JPEG, PNG, BMP e TGA; consultare Formati di file immagine per ulteriori dettagli.

Se si desidera pubblicare online le immagini, utilizzare il formato JPEG per ottenere file di dimensioni minori. Per mantenere elevata la qualità dell'immagine durante una stampa, utilizzare i formati TIFF, PNG o BMP. Il formato TIFF viene comunemente utilizzato per la stampa professionale.

Il salvataggio delle immagini con tavolozza (colori indicizzati come GIF) non è supportato. PhotoZoom Pro 5 converte queste immagini in scala di grigi o colori true color RGB.

Nuovo batch
L'elaborazione di un batch offre la possibilità di ridimensionare contemporaneamente più immagini (un batch, appunto). Supponiamo che sia necessario ridimensionare un centinaio di immagini. Grazie alla funzione di elaborazione batch, non è più necessario aprire un'immagine per volta. È possibile selezionare tutte le immagini, specificare una volta sola la dimensione desiderata e PhotoZoom Pro 5 farà il resto.
Consultare la sezione Elaborazione batch per informazioni più dettagliate sull'utilizzo di questa caratteristica.

Anteprima
Consente di aggiornare l'anteprima nella relativa finestra in base alle impostazioni correnti. Notare che se l'opzione "Anteprima automatica" è attiva (condizione tipica), il pulsante "Anteprima" non è attivo. L'anteprima automatica consente di visualizzare le modifiche in tempo reale dell'immagine nel momento in cui viene cambiata un'impostazione. A volte può risultare utile disattivare l'anteprima automatica (ad esempio in sistemi lenti). In tal caso, deselezionare l'opzione "Anteprima automatica" dal menu "Visualizza". È quindi possibile utilizzare il pulsante "Anteprima" quando si desidera visualizzare con PhotoZoom Pro 5 un'anteprima con le modifiche apportate.

Preferenze
Consente di aprire la finestra Preferenze per la configurazione delle impostazioni.

Aiuto
Consente di aprire il Manuale dell'utente.

Informazioni
Consente di mostrare informazioni sul software e i relativi creatori, numero di versione, informazione di registrazione, collegamenti al sito Web, nonché su licenza e copyright.

Collegamenti
Fornisce un modo semplice per visitare il nostro sito Web (www.benvista.com) e consultare le informazioni aggiornate su PhotoZoom Pro 5 e altri software di nostra creazione. È inoltre possibile avviare altri programmi di BenVista (se installati) o trovare collegamenti diretti alle relative pagine di informazioni sul prodotto.

Il menu

Di seguito è disponibile una panoramica di tutte le voci del menu, comprese le scelte rapide da tastiera disponibili (per Windows e Mac).

File
    Apri (Ctrl+O / Cmd+O) Consente di aprire un'immagine
  Salva con nome... (Ctrl+S / Cmd+S) Consente di specificare nome, destinazione e formato dell'immagine ridimensionata per poi salvarla
  Apri immagine di esempio Consente di aprire l'immagine di esempio di PhotoZoom Pro 5
  Immagini recenti > Consente di riaprire rapidamente un file aperto di recente
  Batch recenti > Consente di riaprire rapidamente un batch salvato di recente
  Esci (Solo Windows) Consente di uscire dal programma

Modifica
 Annulla (Ctrl+Z / Cmd+Z) Consente di ripristinare l'ultima modifica
 Ripeti (Ctrl+Y / Cmd+Shift+Z) Consente di ripetere l'ultima modifica
 Seleziona Consente di attivare lo strumento selezione
 Ritaglia Consente di eliminare la parte selezionata dell'immagine (questa opzione è attiva solo in seguito a una selezione)
 Deseleziona (Ctrl+D / Cmd+D) Consente di deselezionare la parte attualmente selezionata dell'immagine (questa opzione è attiva solo quando viene eseguita una selezione)
 Capovolgi orizzontalmente Consente di capovolgere l'immagine orizzontalmente (la sinistra diventa destra)
 Capovolgi verticalmente Consente di capovolgere l'immagine verticalmente (la parte superiore diventa la parte inferiore)
 Ruota a sinistra Ruota l'immagine di 90 gradi in senso anti orario
 Ruota a destra Ruota l'immagine di 90 gradi in senso orario
 Ripristina immagine Consente di rimuovere tutte le modifiche applicate a partire dall'apertura dell'immagine (impossibile annullare questa operazione)

Batch (fare anche riferimento a Elaborazione batch)
 Nuovo Consente di aprire un nuovo batch vuoto
 Apri Consente di aprire un batch salvato in precedenza
 Salva Consente di salvare il batch corrente
 Salva con nome... Consente di salvare il batch corrente con un altro nome
 Chiudi Consente di chiudere il batch corrente
 Aggiungi immagini... Consente di aggiungere una o più immagini al batch
 Aggiungi cartella... Consente di aggiungere una cartella al batch
 Aggiungi immagine al batch Aggiunge al batch l'immagine correntemente aperta
 Seleziona tutto (Ctrl+A / Cmd+A) Consente di selezionare tutte le immagini attualmente presenti nel batch
 Non selezionare Consente di deselezionare tutte le immagini attualmente presenti nel batch
 Rimuovi selezionati Consente di rimuovere le immagini selezionate dal batch
 Esegui (F5) Consente di avviare il batch con le impostazioni specificate

Opzioni
  Lingua >
 Consente di selezionare una lingua per l'utilizzo di PhotoZoom Pro 5
 Preferenze... Consente di aprire la finestra Preferenze

Visualizza
 Anteprima Consente di aggiornare l'anteprima nella relativa finestra in base alle impostazioni correnti. Questa opzione è disponibile solo se l'opzione "Anteprima automatica" (presente nello stesso menu) è disattivata.
 Anteprima automatica (attiva o non attiva) Una volta attivata, la finestra di anteprima verrà aggiornata dopo ogni modifica apportata
Se disattivata, sarà possibile visualizzare le modifiche facendo clic sul pulsante "Anteprima" o sull'opzione del menu.
 Barra degli strumenti > Consente di impostare le dimensioni della barra degli strumenti (disabilitata, piccola, media o grande)
