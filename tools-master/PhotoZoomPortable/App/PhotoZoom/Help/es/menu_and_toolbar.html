Menú y Barra de Herramientas

Encontrarás los siguientes botones en la barra de herramientas:

Abrir
Abre una imagen existente en la ventana de previsualización.

Guardar
Guarda la imagen redimensionada, después de permitirte elegir la localización y especificar un nombre y formato de archivo. Se soportan los formatos de archivo más comunes como TIFF, JPEG, PNG, BMP y TGA. Consulta Formatos de Archivo de Imagen para más detalles.

Si quieres publicar tus imágenes online, usa el formato JPEG para así conseguir el tamaño de archivo más pequeño. Si lo que quieres es imprimirlas, usa los formatos TIFF, PNG o BMP para mantener la mayor calidad de imagen posible. El formato TIFF es normalmente usado para impresiones profesionales.

El guardado de imágenes de paleta (de colores indexados, como el formato GIF) no está soportado. PhotoZoom Pro 5 convertirá dichas imágenes a escala de grises o a color verdadero RGB.

Nuevo Lote
El procesamiento por lotes te ofrece la posibilidad de redimensionar múltiples imágenes de una sola pasada (un lote). Simplemente di que necesitas redimensionar miles de imágenes a un determinado tamaño: gracias a la característica del procesamiento por lotes no tendrás que abrir las imágenes de una en una, simplemente selecciona todas las imágenes, especifica el tamaño y PhotoZoom Pro 5 hará el resto.
Consulta el capítulo Procesamiento por Lotes para una explicación más detallada de cómo usar esta característica.

Previsualizar
Actualiza la previsualización de la ventana de previsualización, basándose en los ajustes actuales. Ten en cuenta que cuando la "Auto Previsualización" está activada (que normalmente suele ser ese el caso), el botón "Previsualizar" estará desactivado. La Auto Previsualización te asegura que la imagen es actualizada inmediatamente cuando se hace algún cambio. Algunas veces puede ser útil desactivar la auto previsualización (por ejemplo, en equipos antiguos). Para hacer, puedes deseleccinar la opción de "Auto Previsualización" desde el menú "Ver". Así podrás sar el botón "Previsualizar" when quieras que PhotoZoom Pro 5 te muestre una previsualización basada en los cambios que hayas hecho.

El Menú

Más abajo encontrarás una vista general de todos los elementos del menú, incluyendo accesos directos del teclado (tanto para Windows como para Mac) cuando estén disponibles.

Archivo
    Abrir (Ctrl+O / Cmd+O) Abre una imagen
  Guardar como... (Ctrl+S / Cmd+S) Te permite especificar un nombre, una localización y un formato para la imagen redimensionada, y después guardarla
  Abrir la imagen de ejemplo Abre la imagen de ejemplo de PhotoZoom Pro 5
  Imágenes Recientes > Reabre rápidamente un archivo abierto recientemente
  Lotes Recientes > Abre rápidamente un lote guardado recientemente
  Salir (sólo Windows) Cierra el programa

Edición
 Deshacer (Ctrl+Z / Cmd+Z) Deshace el último cambio
 Rehacer (Ctrl+Y / Cmd+Shift+Z) Rehace el último cambio
 Seleccionar Activa la herramienta de selección
 Recortar Elimina la parte de la imagen seleccionada (esta opción solo está activada cuando se hecho una selección)
 Deseleccionar (Ctrl+D / Cmd+D) Deselecciona la parte de la imagen seleccionada (esta opción sólo está activada cuando se ha hecho una selección)
 Volteo Horizontal Vuelca la imagen horizontalmente (la izquierda se vuelve la derecha)
 Volteo Vertical Vuelca la imagen verticalmente (la parte de abajo se vuelve parte de arriba)
 Rotar a la Izquierda Rota la imagen 90 grados a la izquierda
 Rotar a la Derecha Rota la imagen 90 grados a la derecha
 Restaurar Imagen Rechaza todos los cambios aplicados a la imagen desde que fue abierta (esto no puede deshacerse)

Lote (ver también Procesamiento por Lotes)
 Nuevo Abre un lote nuevo y vacío
 Abrir Abre un lote guardado previamente
 Guardar Guarda el lote actual
 Guardar Como... Guarda el lote actual con un nombre diferente
 Cerrar Cierra el lote actual
 Añadir Imágenes... Añade una o más imágenes al lote
 Añadir Carpeta... Añade una carpeta al lote
 Añadir Imagen al Lote Añade la imagen actualmente abierta al lote
 Seleccionar Todo (Ctrl+A / Cmd+A) Selecciona todas las imágenes del lote
 Deseleccionar Todo Deselecciona todas las imágenes del lote
 Eliminar selección Elimina las imágenes seleccionadas del lote
 Ejecutar (F5) Abre el lote con los ajustes establecidos
