Hilfe und Updates

Weitere Informationen sind auf unserer Webseite unter der Kategorie Häufig gestellte Fragen verfügbar.

Sie können technischen Support erhalten, indem Sie uns online über unser Kontaktformular ansprechen. Falls Sie ein Problem melden möchten oder eine Beschwerde haben, beschreiben Sie die Situation bitte so klar wie möglich. Beziehen Sie Information mit ein, die relevant sein könnte, wie Ihren Freischaltcode oder die Seriennummer, die Versionsnummer von Ihrer PhotoZoom Pro 5 Installation (diese finden Sie unter "Info" im Programm), Details von Ihrer Systemkonfiguration (z.B. Betriebssystemversion) und eine funktionierende E-Mail-Adresse, damit wir antworten können.

Wir veröffentlichen oft Softwareupdates, die Verbesserungen und Fehlerbehebungen beinhalten. Diese Updates sind kostenlos und wir empfehlen, immer die neueste verfügbare Version zu benutzen.

Die neuste Version von PhotoZoom Pro 5 kann jederzeit von unserer Webseite heruntergeladen werden. Sie können mit der Software überprüfen, ob eine neuere Version verfügbar ist, indem Sie auf "Suche nach Updates" im Hilfemenü klicken.
