Installazione per Macintosh

Trascinare il file "PhotoZoom Pro 5" nella cartella applicazioni o nella posizione in cui si desidera installare PhotoZoom Pro 5.


I plug-in Adobe® Photoshop® (automazione ed esportazione)

Oltre a essere utilizzabile come applicazione autonoma, PhotoZoom Pro 5 può essere utilizzato come plug-in di Automazione e di Esportazione con Adobe Photoshop e con altri software compatibili con Photoshop.

Prima di installare il plug-in di Automazione e il plug-in di Esportazione, sfogliare la cartella "Plug-ins" di Photoshop. Questa cartella si trova nella stessa cartella in cui è stato installato Adobe Photoshop.

Per installare del plug-ins, copiare il file "PhotoZoom Pro 5.plugin" nella cartella "Plug-ins" di Photoshop.

Dopo aver installato i plug-in, PhotoZoom Pro 5 viene elencato in Photoshop nel menu "File" > "Automatizza" e "File" > "Esporta". È necessario riavviare Photoshop per assicurare il riconoscimento dei plug-in.

Disinstallazione

Per disinstallare PhotoZoom Pro 5 manualmente, eliminare tutti i file copiati durante l'installazione manuale. È necessario eliminare anche il file "Preferenze PhotoZoom Pro 5". Questo file è presente nella cartella Preferenze del vostro sistema, che potete trovare andando nel menu "Vai" del Finder, selezionando l'opzione "Vai alla cartella..." e inserendo :

 ~/Library/Preferences/

L'eliminazione dei file delle preferenze implica la rimozione di qualsiasi codice di sblocco inserito in precedenza.
