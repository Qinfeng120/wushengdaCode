Enregistrement

Si vous avez acheté PhotoZoom Pro 5, vous pouvez le déverrouiller en entrant la clé de déverrouillage fournie avec le logiciel.

Si vous n'avez pas déverrouillé le logiciel, la fenêtre d'entrée de la clé sera affichée automatiquement au lancemet du logiciel (version autonome ou plug-in).



Certaines licenses nécessitent un nom et une adresse e-mail en plus de la clé de déverrouillage. Dans ce cas, ces informations sont clairement spécifiées avec la clé que vous avez reçue. Vérifiez que vous rentrez toutes les informations exactement comme elles vous ont été fournies.

Si vous avez acquis ce logiciel par une mise à jour, il vous sera peut-être demandé de préciser les informations de déverrouillage. Dans ce cas, remplissez exactement toutes les zones.

Il vous suffit d'entrer la clé une seule fois pour déverrouiller le logiciel de façon permanente. Vous n'aurez plus à entrer cette clé lors des mises à jour futures.

Attention : aussi longtemps que vous n'aurez pas déverrouillé le logiciel, il s'exécutera en mode essai. Ce mode entraîne une limitation : les images que vous sauvegarderez comporteront un filigrane. Lorsque le logiciel est utilisé en mode essai, vous pouvez passer en mode production en cliquant sur "déverrouiller" dans le menu Aide.

Si vous avez acheté le logiciel par Internet et reçu votre clé par courriel, nous vous recommandons de copier-coller cette information afin d'éviter les erreurs de frappe.
