Installation for Windows

To install PhotoZoom Pro 5, run the Setup file or insert the CD. If your CD-drive's Autoplay feature has been disabled, browse to the CD and run the Setup file manually.

After reading the license agreement select the option "I accept the agreement" to activate the "Next" button:



During the installation, you can customize where and how PhotoZoom Pro 5 will be installed onto your system. Changing the default settings of the installer is not a necessity: simply following the windows by clicking "Next" and eventually "Finish" will be suitable in most cases.
When the installation is finished, start PhotoZoom Pro 5 by double clicking the icon on the desktop or in the start menu:

The Adobe® Photoshop® plug-ins (Automation & Export)

Besides running as a stand-alone application, PhotoZoom Pro 5 can also be used as Adobe Photoshop compatible Automation and Export plug-in. The plug-ins will only be installed if Photoshop is already present on your system when you start the installation of PhotoZoom Pro 5. The installer automatically detects where to install the plug-ins. You can also specify a custom folder to install the plug-ins into, but in that case make sure Photoshop is configured to look for plug-ins in that folder.

Once you have installed the plug-ins, PhotoZoom Pro 5 will be listed in Photoshop under "File" > "Automate" and "File" > "Export". You will need to restart Photoshop to make sure it recognizes the plug-in.

Other host programs

In Adobe® Photoshop® Elements, the Automation plug-in is either listed under "File" > "Automation Tools" or "File" > "Automate", depending on your version. The Export plug-in is accessible via "File" > "Export"

In Corel® PHOTO-PAINT, the Export plug-in is accessible via "File" > "File Format Plug-ins" > "Export".

In Corel® PaintShop Pro (Photo), the Export plug-in is accessible via "File" > "Export" > "Plugins".


PhotoZoom Pro 5 can be used as a plug-in for various host programs. See the system requirements for a complete overview.
