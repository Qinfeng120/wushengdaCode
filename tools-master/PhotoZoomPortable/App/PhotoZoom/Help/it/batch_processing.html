Elaborazione batch

Questa funzione consente di ridimensionare più immagini nello stesso momento. È possibile applicare le stesse impostazioni a tutte le immagini oppure utilizzare impostazioni diverse per immagini o gruppi di immagini differenti. Selezionare l'opzione desiderata e PhotoZoom Pro 5 si occuperà del resto.

Facendo clic sul pulsante "Nuovo batch", il menu relativo al batch verrà visualizzato nella parte inferiore dello schermo:

1) Scelta delle immagini da elaborare
Il pulsante "Aggiungi immagini..." consente di aggiungere una o più immagini da una cartella al batch. Per aggiungere una cartella completa di immagini, utilizzare il pulsante "Aggiungi cartella...". Selezionando l'opzione "Includi sottocartelle" prima di selezionare una cartella, verranno aggiunte tutte le immagini presenti nelle sottocartelle della cartella.

2) Elenco immagini
Elenco dei file contenuti nel batch insieme a tutti i dettagli rilevanti: dimensione di origine, informazioni di ridimensionamento, metodo di ridimensionamento e dimensione di destinazione. Se le impostazioni vengono modificate (se si cambia, ad esempio, il metodo di ridimensionamento), le informazioni contenute nell'elenco vengono modificate di conseguenza. Tali modifiche si applicano soltanto alle immagini dell'elenco che sono state selezionate. È possibile selezionare immagini singole, gruppi separati o l'elenco completo.

È possibile modificare la selezione in diversi modi:

  • Per aggiungere immagini singole alla selezione, tenere premuto il tasto Ctrl (su Windows) o Cmd (su Mac) e fare clic sull'immagine che si desidera aggiungere

  • Per selezionare un gruppo di immagini, fare clic su un'immagine, quindi su un'altra tenendo premuto il tasto MAIUSC. Entrambe le immagini e tutte quelle incluse nel messo verranno selezionate

3) Pulsanti di selezione
Seleziona tutto: consente di selezionare tutte le immagini dell'elenco
Deseleziona tutto: consente di deselezionare tutte le immagini selezionate
Rimuovi selezione: consente di rimuovere tutte le immagini selezionate dall'elenco

4) Impostazioni di ridimensionamento
In questa sezione è possibile specificare le nuove dimensioni e le impostazioni del metodo di ridimensionamento in relazione alle immagini contenute nel batch. Per informazioni dettagliate sulle impostazioni, fare riferimento al capitolo Informazioni sul programma (paragrafi 2 e 3) e Affinamento. Le modifiche vengono applicate solo sui file selezionati.

Nota: se vengono selezionate più immagini, alcuni valori nella sezione Nuova dimensione potrebbero non essere visibili. Ciò dipende dalle dimensioni delle immagini selezionate, che possono variare da un'immagine all'altra, nonché dal modo in cui tali dimensioni sono state specificate (ad esempio: pixel o percentuali possono fare la differenza). Di conseguenza, potrebbe essere impossibile visualizzare un valore che sia corretto per ognuna delle immagini selezionate. In questo caso PhotoZoom Pro 5 lascia il valore vuoto. Tuttavia, è possibile trovare tutte le informazioni in ognuna delle immagini singole nel relativo elenco del menu Batch.

5) Esegui
Dopo aver definito tutte le impostazioni, fare clic sul pulsante "Esegui". Viene quindi visualizzata la seguente finestra:



Per modificare la cartella di destinazione, fare clic sul pulsante "Sfoglia...". Se viene specificata una cartella che ancora non esiste, PhotoZoom Pro 5 ne richiede la creazione.

In "Formato di destinazione" è possibile salvare le immagini come file TIFF, JPEG, PNG, BMP o TGA. Il formato TIFF è utilizzato di frequente poiché supporta un'ampia gamma di funzionalità di immagine (vedere anche la sezione relativa ai formati di file di immagine). Per i file più piccoli, scegliere il formato JPEG.

Infine, è possibile stabilire la procedura con cui PhotoZoom Pro 5 consente di salvare un file quando esiste già un altro file con lo stesso nome nella cartella di destinazione.

Pronti per iniziare? Fare clic su "Avvia batch" per avviare l'elaborazione:



La finestra di stato indica una stima del tempo necessario a PhotoZoom Pro 5 per completare il processo. Tale stima viene costantemente aggiornata. In base alle impostazioni e al numero delle immagini, il processo potrebbe richiedere del tempo. Inoltre, l'elaborazione del batch utilizza l'alimentazione del computer e, pertanto, potrebbe compromettere le prestazioni delle altre applicazioni in esecuzione nello stesso momento. Per questa ragione, è sempre possibile sospendere il processo e ripristinarlo in seguito, in un momento più appropriato.
