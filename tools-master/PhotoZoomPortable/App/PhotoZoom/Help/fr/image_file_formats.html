Formats d'images

Ce chapitre présente les formats d'images supportés par PhotoZoom Pro 5 et leurs spécifications.

Cette présentation rapide ne concerne que la version autonome de PhotoZoom Pro 5. Les modules complémentaires Automatisation et Exportation permettent de traiter un plus grand nombre de formats de fichiers, y compris PSD, EPS et les différents formats RAW. Le filtre d'Automatisation supporte également les images au format HDR. Lorque vous utilisez le filtre d'Exportation, vous ne pouvez enregistrer les résultats que dans un format autorisé (TIFF par exemple). Cette limitation ne s'applique pas au filtre d'automatisation.

1: Le format TIFF supporte des modes de compression "sans perte" ou "avec perte" (basé sur le JPEG). Lorsque vous sauvegardez une image en format TIFF, vous pouvez indiquer quel type de compression vous souhaitez utiliser.

2: Ce format supporte un mode de compression. PhotoZoom Pro 5 peut ouvrir de telles images mais les sauvegardera en mode non compressé.

3: Le format GIF ne supporte que les couleurs par palettes et il est généralement utilisé pour des images de basse qualité (256 couleurs). PhotoZoom Pro 5 peut ouvrir des images en format GIF, les traite comme des images RVB, mais ne peut les sauvegarder ensuite au format GIF.
