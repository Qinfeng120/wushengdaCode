Bilddateiformate

Dieses Kapitel beinhaltet eine Übersicht über Bildformate, die von PhotoZoom Pro 5 unterstützt werden, und ihren Spezifikationen.

Beachten Sie, dass dieser Überblick die selbstständige Version von PhotoZoom Pro 5 betrifft. Die Versionen Automation PlugIn und Export PlugIn können eine größere Palette an Bildformaten verarbeiten, einschließlich PSD, EPS und Raw Formate. Das Automation PlugIn unterstützt außerdem HDR Bilder. Wenn Sie das Export PlugIn nutzen, können Sie Ihre Dateien nur in einem unterstützten Format (wie zum Beispiel TIFF) abspeichern. Solche Einschränkungen gibt es beim Automation PlugIn nicht.

1: Das TIFF Format unterstützt sowohl verlustfreie als auch verlustreiche (JPEG-basierte) Kompression. Wenn Sie Bilder als TIFF speichern, können Sie auswählen, welche Kommpressionsart Sie bevorzugen.

2: Das Format unterstützt Lauflängenkompression. PhotoZoom Pro 5 kann sie öffnen, speichert aber unkomprimierte Bilder.

3: Das GIF Format unterstützt nur Farbpaletten und wird normalerweise für Bilder mit geringer Qualität (256 Farben) gebraucht. PhotoZoom Pro 5 kann GIF Bilder öffnen (und behandelt sie dabei wie RGB), speichert sie aber nicht als GIF.
