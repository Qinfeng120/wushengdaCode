Stapelverarbeitung

Mit dieser Funktion ist es möglich die Größe von mehreren Bildern gleichzeitig anzupassen. Sie können die gleichen Einstellungen bei allen Bildern anwenden, oder verschiedene Einstellungen für verschiedene Bilder(gruppen) verwenden. Wählen Sie einfach Ihre Einstellung und PhotoZoom Pro 5 erledigt den Rest.

Wenn Sie den Button "Neue Stapelverarbeitung" drücken, öffnet sich am unteren Ende des Bildschirms das Stapelverarbeitungsmenü:

1) Wählen Sie die Bilder aus, die Sie bearbeiten möchten
Mit dem "Bilder hinzufügen..."-Button können Sie ein oder mehrere Bilder von einem Ordern zu Ihrer Stapelverarbeitung hinzufügen. Benutzen Sie den "Ordner hinzufügen..."-Button, um einen ganzen Ordner mit Bilder hinzuzufügen. Wenn Sie die "Unterordner miteinbeziehen"-Option aktivieren, bevor Sie einen Ordner auswählen, werden alle Bilder, die sich in Unterordnern befinden, ebenfalls hinzugefügt.

2) Bilderliste
Hier sind Ihre Stapeldateien zusammen mit allen relevanten Details aufgelistet: Eingabegröße, Informationen zur Vergrößerung, Methode für Vergrößerung und Ausgabegröße. Wenn Sie die Einstellungen verändern (zum Beispiel die Methode für Vergrößerung), verändert sich die Information in der Liste dementsprechend. Die Änderungen, die Sie machen, gelten nur für die ausgewählten Bilder in Ihrer Liste. Sie können einzelne Bilder, separate Gruppen oder die gesamte Liste auswählen.

Auswahländerungen können auf unterschiedliche Weise durchgeführt werden:

  • Um einzelne Bilder der Auswahl hinzuzufügen, halten Sie Ctrl/Strg (Windows) oder Cmd (Mac) gedrückt und klicken Sie auf die Bilder, die Sie hinzufügen möchten

  • Um eine Gruppe von Bildern auzuwählen, klicken Sie auf ein Bild und anschließend auf ein anderes Bild, während Sie die Shift-Taste gedrückt halten. Die zwei Bilder und jegliche Bilder dazwischen werden dadurch markiert

3) Auswahlbuttons
Alle auswählen: Wählt alle Bilder aus der Bilderliste aus
Keine auswählen: Deselektiert alle Bilder, die markiert wurden
Auswahl entfernen: Entfernt alle ausgewählten Bilder aus der Bilderliste

4) Einstellungen zur Größenveränderung
Hier bestimmen Sie die neuen Größen und Einstellungen zur Vergrößerungsmethode für die Bilder im Stapel. Für ausführliche Informationen zu diesen Einstellungen schauen Sie bitte in die Kapitel Programmerklärung (Punkte 2 und 3) und Feinabstimmung. Die Änderungen, die Sie machen, werden nur auf die ausgewählten Dateien angewendet.

Bitte beachten Sie: wenn mehrere Bilder ausgewählt sind, sind möglicherweise einige Werte in dem Feld für "Neue Größe" nicht sichtbar. Dies hängt sowohl von der Größe der ausgewählten Bilder ab, welche von einander abweichen können, als auch davon, wie die Größen bestimmt werden (zum Beispiel kann es einen Unterschied machen, ob sie in Pixel oder Prozent angegeben sind). Desahlb ist es nicht möglich, einen Wert anzuzeigen, der für jedes der ausgewählten Bilder korrekt ist. In so einem Fall lässt PhotoZoom Pro 5 den Wert leer. Trotzdem können Sie immer noch alle Information zu den einzelnen Bilder in der Bilderliste des Stapelverarbeitungsmenüs finden.

5) Ausführen
Wenn alle Einstellungen gemacht wurden, klicken Sie auf den "Ausführen"-Button. Das folgende Fenster erscheint dann:



Wenn Sie den Ausgabeordner ändern möchten, klicken Sie auf den "Durchsuchen..."-Button. Wenn Sie einen Ordner bestimmen, der noch nicht existiert, wird PhotoZoom Pro 5 nachfragen, ob dieser Ordner erstellt werden soll.

Unter "Ausgabeformat" können Sie auswählen, ob Sie Ihre Bilder als TIFF-, JPEG-, PNG-, BMP- oder TGA-Dateien speichern möchten. Das TIFF-Format wird häufig verwendet, da es eine breite Auswahl an Bildfunktionen unterstützt (siehe auch Bilddateiformate). Wählen Sie das JPEG-Format um möglichst kleine (und damit am meisten komprimierte) Dateien zu erhalten.

Schließlich können Sie wählen, was PhotoZoom Pro 5 tun soll, wenn eine Datei gespeichert werden soll, in diesem Ordner jedoch bereits eine andere Datei mit demselben Namen existiert.

Sind Sie bereit? Klicken Sie auf "Stapelverarbeitung starten", um mit der Bearbeitung zu beginnen:



Dieses Ablauf-Fenster macht eine Schätzung für die Zeit, die PhotoZoom Pro 5 erwartet, um den Auftrag zu erledigen. Die Schätzung wird durchgehend aktualisiert. Abhängig von den Einstellungen und der Anzahl der Bilder kann der Prozess einige Zeit dauern. Beachten Sie bitte auch, dass die Stapelverarbeitung Leistung von Ihrem Computer benötigt, was eventuell einen negativen Einfluss auf die Ausführung anderer Programme haben kann, die zur gleichen Zeit durchgeführt werden. Aus diesem Grund können Sie den Prozess immer anhalten und ihn später zu einem günstigeren Zeitpunkt fortsetzen.
