Installation for Macintosh

Drag and drop the file "PhotoZoom Pro 5" to the Applications folder, or wherever you want to install PhotoZoom Pro 5.


The Adobe® Photoshop® plug-ins (Automation & Export)

Besides running as a stand-alone application, PhotoZoom Pro 5 can also be used as Automation and Export plug-in with Adobe Photoshop and Photoshop-compatible software.

Prior to installing the Automation and Export plug-ins, browse to Photoshop's "Plug-ins" folder. This folder is located in the folder where you have installed Adobe Photoshop.

To install the plug-ins, copy the file "PhotoZoom Pro 5.plugin" into Photoshop's "Plug-ins" folder.

Once you have installed the plug-ins, PhotoZoom Pro 5 will be listed in Photoshop under "File" > "Automate" and "File" > "Export". You will need to restart Photoshop to make sure it recognizes the plug-ins.

If you are using Photoshop® Elements, the Automation plug-in is either listed under "File" > "Automation Tools" or "File" > "Automate", depending on your version. The Export plug-in is accessible via "File" > "Export".


Uninstalling

To uninstall PhotoZoom Pro 5 manually, delete all the files which you copied during the manual installation. You should also delete the "PhotoZoom Pro 5 Preferences" file. This file is located in the Preferences folder of your system, which you can locate by going to the Finder's "Go" menu, selecting the option "Go to Folder..." and entering:

 ~/Library/Preferences/

Note that deleting the preferences file will also remove any previously entered unlock code.
