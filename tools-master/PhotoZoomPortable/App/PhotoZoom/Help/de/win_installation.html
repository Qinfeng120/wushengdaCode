Installation für Windows

Um PhotoZoom Pro 5 zu installieren, führen Sie die Setup-Datei aus oder legen Sie die CD ein. Falls die Autoplay-Funktion ihres CD-Laufwerks nicht aktiviert ist, durchsuchen Sie die CD und führen Sie die Setup-Datei manuell aus.

Nachdem Sie die Lizenzbestätigung gelesen haben, akzeptieren Sie diese, um den "Weiter"-Button zu aktivieren:



Während der Installation können Sie festlegen, wo und wie PhotoZoom Pro 5 auf Ihrem System installiert werden soll. Es ist nicht nötig die Standardeinstellungen zu verändern; in den meisten Fällen ist es passend, den Anweisungen mit "Weiter" zu folgen, zum Beenden der Installation klicken Sie bitte abschließend auf "Beenden".

Die Adobe® Photoshop® PlugIns (Automation & Export)

Außer als selbstständige Anwendung kann PhotoZoom Pro 5 auch als zu Adobe Photoshop kompatibles Automation und Export PlugIn verwendet werden. Die PlugIns werden nur installiert, wenn Photoshop bereits auf Ihrem System vorhanden ist, wenn Sie die Installation von PhotoZoom Pro 5 starten. Das Installationsprogramm erkennt dann automatisch, wohin die PlugIns installiert werden müssen. Sie können auch einen benutzerdefinierten Ordner für die Installation der PlugIns angeben. In diesem Fall muss aber sichergestellt werden, dass Photoshop so konfiguriert ist, dass auch in diesem Ordner nach PlugIns gesucht wird.

Sobald Sie die PlugIns installiert haben, wird PhotoZoom Pro 5 in Photoshop unter "Datei" > "Automatisieren" und "Datei" > "Exportieren" aufgelistet. Sie müssen Photoshop neu starten, um sicher zu gehen, dass die PlugIns erkannt werden.

Andere Host-Programme

In Adobe® Photoshop® Elements ist das Automatisierungs plug-in entweder unter "Datei" > "Automatisierungswerkzeuge" oder "Datei" > "Automatisieren" gelistet, abhängig von Ihrer Version. Auf das Exportieren plug-in kann über "Datei" > "Exportieren" zugegriffen werden.

In Corel® PHOTO-PAINT ist das Exportieren plug-in über "Datei" > "Dateiformat Plug-ins" > "Exportieren" zugänglich.

In Corel® PaintShop Pro (Photo) ist das Exportieren plug-in über "Datei" > "Exportieren" > "Plugins" erreichbar.


PhotoZoom Pro 5 kann als ein plug-in für verschiedene Host-Programme genutzt werden. Für eine Gesamtübersicht betrachten Sie die Systemanforderungen.
