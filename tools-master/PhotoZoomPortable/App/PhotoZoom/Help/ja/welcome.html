ようこそ

BenVistaが開発したS-Splineテクノロジーは、PhotoZoom Pro 5に世界最高の画像拡大/縮小革命をもたらします。
PhotoZoom Pro 5を使うことで、従来技術では不可能であった、エッジの荒さ、ぼやけ、不自然な作り物っぽさ等のない、自然で鮮明な拡大画像を作成することが出来ます。パワフルでありながらインターフェースも至ってシンプル、かつ使いやすいのが特徴です。

このソフトは、DTP、工業用画像処理、医療アプリケーション、セキュリティ、拡大プリント、デジタル写真撮影、オンライン出版、インターネット上のイメージ加工、などに携わる人に適しています。

PhotoZoom Pro 5の主な機能:
  • BenVistaが開発したS-Splineテクノロジーは、最新技術S-Spline Maxを含みます。
  • 高精細リサイズ技術
  • 入力イメージにより、方式を選択可能です。
  • 高度な調整ツール：独自のリサイズ方法のプリセットを作成します。
  • プロファイルのサイズ変更：お気に入りのコンピレーションでシングルクリックサイズ設定、リサイズ方法のプリセットを選択してください。
  • 新たにJPEG圧縮アーティファクト / ノイズ低減を装備！
  • 自然なイメージにするため、フィルムグレイン機能を搭載
  • 規格外の拡大：　最大100万X100万ピクセル
  • マルチプロセッサ対応
  • グラフィックスプロセッシングユニット (GPU)サポート:最大5倍速の画像処理 (システムにより異なります)
  • バッチ変換：　一つの操作で大量のイメージをリサイズ
  • 卓越したトリミングツール
  • スタンドアローンアプリとして、また 　Adobe PhotoshopのAutomationやExport plug-inとして使用可能 (システム要件)
  • メタデータやカラープロファイルを保持したまま出力可能
  • 16bit イメージサポート
  • 32bitチャンネルイメージサポート（Automation plug-inインポートを使用時）
  • レイヤーサポート（Automation plug-inインポートを使用時）
  • 入力対応フォーマットが充実（Automation plug-inインポートを使用時）
  • Windows / OS X / Adobe Photoshop に完全互換しています。
  • スタンドアローン / プラグインアプリ共に32/64bit環境に完全対応！


    • Mac:
  • ユニバーサルスタンドアロンアプリケーションおよびプラグイン：　PowerPCおよびインテルベースのシステムで利用可能
