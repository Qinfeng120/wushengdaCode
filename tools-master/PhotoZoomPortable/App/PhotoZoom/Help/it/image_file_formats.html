Formati di file di immagine

In questa sezione viene fornita una panoramica dei formati immagine supportati da PhotoZoom Pro 5, nonché le relative specifiche.

Questa panoramica fa riferimento alla versione autonoma di PhotoZoom Pro 5. Le versioni di plug-in relative ad automazione ed esportazione possono elaborare una gamma di formati di immagini più ampia, inclusi i formati PSD, EPS e non elaborato. Inoltre, il plug-in relativo all'automazione supporta le immagini HDR. Utilizzando il plug-in di esportazione, è possibile salvare i file solo in un formato supportato (come TIFF). Il plug-in relativo all'automazione non ha alcuna restrizione.

n/d: opzione non supportata dal formato immagine.

1: Il formato TIFF supporta compressioni con e senza perdita di dati (basate su JPEG). Durante il salvataggio delle immagini in formato TIFF, è possibile scegliere il tipo di compressione desiderato.

2: Il formato supporta la compressione run length. PhotoZoom Pro 5 è in grado di aprire tali formati, ma salva le immagini non compresse.

3: Il formato GIF supporta solo i colori della tavolozza ed è solitamente utilizzato per immagini di bassa qualità (256 colori). PhotoZoom Pro 5 consente di aprire immagini GIF (ed elaborarle come RGB), ma non di salvarle come GIF.
