メニューおよびツールバー

メインメニューの上部には主要機能に簡単にアクセスできるようにツールバーが用意されています。

開く
対象となるファイルを開きます。

保存
リサイズした画像を保存します。保存先を選択した後、ファイル名およびフォーマットを指定します。TIFF、JPEG、PNG、BMP、TGAなどの主要フォーマットをサポートしています。 詳細は、画像ファイルフォーマットを参照して下さい。

変更後のファイルをオンライン上で使用する場合、フォーマット形式にJPEGを指定するとファイルサイズを小さく出来ます。変更後のファイルを印刷する場合、TIFF、PNG、または、BMPを指定すると画質を落とさずに保存できます。特にTIFFフォーマットは、印刷関連の業界で広く使われています。

パレットを使用した画像フォーマット（GIFなどのインデックスカラー）はサポートしていません。PhotoZoom Pro 5はそれらの画像をグレースケールまたはトゥルーカラーRGBに変換します。

新規バッチ
バッチ処理は、一回の処理で複数画像のリサイズを行います。例えば、1000画像をある一定のサイズに変更する場合、画像を一つずつ処理するのではなく、それらをバッチに追加して一回の処理で全ての画像をリサイズすることが出来ます。
詳細は、バッチ処理を参照して下さい。

プレビュー
現在の設定をプレビュー画面に反映します。自動プレビューが選択されている場合、プレビューボタンは非表示になっています。自動プレビューを選択している場合、設定変更はすぐにプレビュー画面に反映されます。自動プレビュー は、処理の遅いPCなどでは非選択にした方が良い場合があります。自動プレビューを非選択にするには、メニューの「表示」から「自動プレビュー」をクリックしてチェックをはずします。設定変更をプレビュー画面に反映するにはプレビューボタンをクリックします。

環境設定
さまざまな設定を行うことができる環境設定ウインドウを開きます。

ヘルプ
ヘルプを表示します。

について
本製品に関するバージョン、作成元、ライセンス等の情報を表示します。

メニュー

下記メニュー一覧では、各項目、ショートカット（WindowsおよびMac）等を説明しています。

ファイル
    開く (Ctrl+O / Cmd+O) 画像を開く
  別名で保存... (Ctrl+S / Cmd+S) 画像のファイル名、場所、フォーマットを指定して保存します。
  サンプル画像を開く PhotoZoom Pro 5のサンプル画像を開く
  最近使った画像 > 最近使ったファイルを開く
  最近使ったバッチ > 最近保存したバッチを開く
  終了 プログラムを終了する（Windowsの場合）

編集
 元に戻す (Ctrl+Z / Cmd+Z) 最後に行った変更を取り消す
 繰り返す (Ctrl+Y / Cmd+Shift+Z) 最後に行った変更を繰り返す
 選択範囲を指定 選択ツールを有効にする
 トリミング 選択部分を切り抜く（選択範囲が指定されている時のみ有効）
 非選択 (Ctrl+D / Cmd+D) 選択解除は、指定された箇所を解除します。
 水平方向に反転 水平方向に画像を反転する
 垂直方向に反転 垂直方向に画像を反転する
 左に回転 逆時計回りに90度回転
 右に回転 時計回りに90度回転
 画像を元の状態に戻す 全ての変更をキャンセルして画像を開いた時の状態に戻す（この操作を取り消すことは出来ません）

バッチ （バッチ処理を参照）
 新規 新規の（空の）バッチを作成する
 開く 保存されているバッチを開く
 保存 現在のバッチを保存する
 別名で保存 現在のバッチを別名で保存する
 閉じる 現在のバッチを閉じる
 画像を追加 画像をバッチに追加する
 フォルダ追加... バッチにフォルダを追加する
 画像をバッチに追加 現在開いている画像をバッチに追加
 全て選択 (Ctrl+A / Cmd+A) バッチ内の全画像を選択する
 選択解除 バッチ内の全画像を選択解除する
 選択箇所を削除 バッチ内の選択されている画像を削除する
 実行 (F5) 指定した設定でバッチを実行する

オプション
  言語 >
 プログラムで使用する言語を変更する
 環境設定... 環境設定画面を開く

表示
 プレビュー 現在の設定をプレビュー画面に反映する。この設定は自動プレビューが非選択の時のみ有効
 自動プレビュー（有効/無効） この設定が有効の場合、変更後の内容は自動的に反映される
この設定が無効の場合、変更後の内容は、プレビューボタンをクリックした時のみ反映される
 ツールバー > ツールバーのサイズを変更する（非表示、小、中、大）

ヘルプ
 ヘルプ ヘルプを開く
 アップデータを確認... PhotoZoom Pro 5のアップデータがあるかどうか確認する
 購入 株式会社ジャングルの購入ページ ( http://www.junglejapan.com/products/photo/pzp/ ) を表示する
 アンロック... ソフトウェアのロックを解除する（ロックコードの解除を参照）
 サポート 株式会社ジャングルのサポートページ ( http://www.junglejapan.com/support/ ) を表示する
 について 本製品に関するバージョン、作成元、ライセンス等の情報を表示します。
