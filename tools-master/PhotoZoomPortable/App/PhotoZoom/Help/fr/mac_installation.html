Installation pour MacIntosh

Cliquez-déplacez le fichier "PhotoZoom Pro 5" vers le dossier Applications où vers l'endroit où vous souhaitez installer PhotoZoom Pro 5.


Les modules complémentaires "Automatisation" et "Exportation" d'Adobe ® Photoshop ®

PhotoZoom Pro 5 peut être utilisé comme une application autonome ("stand-alone") ou en conjonction avec Adobe Photoshop, comme module complémentaire des fonctions "Automatisation" et "Exportation".

Avant d'installer les modules complémentaires d'automatisation et d'exportation, regardez dans le répertoire "Plug-ins" de Photoshop. Ce répertoire est situé dans le répertoire où vous avez installé Adobe Photoshop.

Pour installer les modules complémentaires, copiez le fichier "PhotoZoom Pro 5.plugin" dans le répertoire "Plug-ins" de Photoshop.

Après avoir installé les modules complémentaires, PhotoZoom Pro 5 apparaîtra lorsque vous sélectionnerez Fichier > Automatisation ou Fichier > Exportation. Vous devrez redémarrer Photoshop pour que ces modules soient pris en compte.

Si vous utilisez Photoshop® Elements, le module complémentaire (plug-in) Automatisation est accessible soit par la commande "Fichier" > "Outils d'automatisation", soit par "Fichier" > "Automatisation", en fonction de votre version. Le module d'Exportatilon est accessible par "Fichier" > "Exportation".


Désinstallation

Pour désinstaller PhotoZoom Pro 5 manuellement, supprimez tous les fichiers que vous avez copiés pendant l'installation. Vous devrez également supprimer le fichier "Préférences" de PhotoZoom Pro 5. Ce fichier est situé dans le dossier Préférences de votre système. Pour le trouver, allez dans le menu "Aller" du Finder, sélectionnez l'option "Aller au dossier..." et entrez :

 ~/Library/Preferences/

Attention : en supprimant le fichier de préférences, vous supprimez également la clé d'enregistrement !
