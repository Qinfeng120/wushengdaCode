Menu et Barre d'outils

Dans la barre d'outils, vous trouverez les boutons suivants :

Ouvrir
Ouvre une image dans la fenêtre de pré-visualisation.

Enregistrer
Sauvegarde l'image redimensionnée après avoir choisi un emplacement de sauvegarde et un nom de fichier. Les formats usuels (TIFF, JPEG, PNG, BMP et TGA) sont supportés. Consultez le chapitre Formats d'image pour plus d'information.

Si vous souhaitez utiliser vos images en ligne, utiliez le format JPEG pour obtenir les fichiers les plus petits. Si vous voulez les imprimer, utilisez les formats TIFF, PNG ou BMP pour conserver le maximum de qualité. Le format TIFF est généralement utilisé pour des images de qualité professionnelle.

La sauvegarde d'images utilisant des palettes (couleurs indexées, telles que le format GIF) n'est pas supportée. PhotoZoom Pro 5 convertira ces images en niveaux de gris ou couleurs (mode RVB).

Nouveau lot
Le traitement par lot permet de redimensionner simultanément plusieurs images. Si vous avez par exemple un millier d'images à redimensionner à la même taille, vous utiliserez le traitement par lot. Cela vous évitera d'ouvrir individuellement chaque image : il vous suffira de les sélectionner, d'indiquer une seule fois la taille de sortie et PhotoZoom Pro 2 fera le reste !
Consultez le chapitre Traitement par lot pour plus d'informations sur cette fonction.

Prévisualisation
Met à jour l'affichage dans la fenêtre de prévisualisation. Si l'option "Prévisualisation auto" du menu "Afficher" est cochée, le bouton "Prévisualisation" sera inactif. L'option "Prévisualisation auto" entraîne automatiquement un rafraîchissement de l'image dans la fenêtre de prévisualisation lors d'une modification des paramètres. Il est parfois utile de désactiver cette possibilité (sur des sytèmes lents par exemple). Il vous suffit de décocher l'option "Prévisualisation auto" du menu "Afficher". Vous utiliserez alors le bouton "Prévisualisation" lorsque vous souhaiterez voir le résutat des modifications apportées aux paramètres.

Préférences
Ouvre la fenêtre "Préférences" où vous pouvez définir vos paramètres par défaut.
