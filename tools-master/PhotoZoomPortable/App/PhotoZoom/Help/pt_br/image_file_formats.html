Formatos de arquivo de imagens

Este capítulo apresenta um resumo dos formatos de imagens suportados pelo PhotoZoom Pro 5, e especificações dessas imagens.

Nota: este resumo se refere à versão dedicada do PhotoZoom Pro 5. As versões do plug-in de Automatização e de Exportação são capazes de processar uma faixa mais ampla de formatos de imagens, inclusive os formatos PSD, EPS e imagem bruta. O plug-in de automatização também suporta imagens HDR. Quando se utiliza o plug-in de Exportação, os arquivos somente podem ser salvos em um formato suportado (como o TIFF). O plug-in de Automatização não apresenta essa limitação.

n/a: Esta opção não é suportada pelo formato da imagem.

1: O formato TIFF suporta ambos os tipos de compressão "lossless" (compactação, sem perda de dados) e "lossy" (compactação com perda de dados) no formato JPEG. Ao salvar as imagens no formato TIFF, o usuário poderá escolher o tipo de compressão que deseja.

2: O formato suporta codificação "run-length". O PhotoZoom Pro 5 é capaz de abrir essas imagens comprimidas, porém as salva como imagens não comprimidas.

3: O formato GIF suporta somente paletas de cores, sendo tipicamente utilizado para imagens de baixa qualidade (256 cores). O PhotoZoom Pro 5 é capaz de abrir imagens no formato GIF (e tratá-las como RGB) mas não as salva no formato GIF.
