Menü und Symbolleiste

Sie werden die folgenden Buttons in der Symbolleiste finden:

Öffnen
Öffnet ein vorhandenes Bild in das Vorschaufenster.

Speichern
Speichert das größenveränderte Bild, nachdem Sie einen Ort ausgewählt und Dateiname und Format bestimmt haben. Gebräuchliche Formate wie TIFF, JPEG, PNG, BMP, und TGA werden unterstützt; für weitere Details siehe Bilddateiformate.

Wenn Sie Ihre Bilder online veröffentlichen möchten, benutzen Sie das JPEG-Format, damit die Dateien möglichst klein sind. Wenn Sie die Bilder drucken möchten, benutzen Sie TIFF, PNG oder BMP um eine so hohe Bildqualität wie möglich beizubehalten. TIFF wird üblicherweise für professionelles Drucken benutzt.

Das Speichern von palettierten Bildern (indizierte Farben, wie z.B. GIF) wird nicht unterstützt. PhotoZoom Pro 5 wird diese Bilder in graustufiges oder vollfarbiges RGB konvertieren.

Neue Stapelverarbeitung
Die Stapelverarbeitung bietet die Möglichkeit die Größe mehrerer Bilder auf einmal anzupassen. Angenommen Sie möchten tausend Bilder an eine bestimmte Größe anpassen; mit der Stapelverarbeitungsfunktion müssen nicht alle Bilder einzeln geöffnet werden, sondern stattdessen können alle Bilder gleichzeitig ausgewählt werden und die gewünschte Größe einmal bestimmt werden.
Lesen Sie im Kapitel Stapelverarbeitung eine detailliertere Beschreibung darüber, wie Sie dieses Feature benutzen können.

Vorschau
Aktualisiert die Vorschau im Vorschaufenster, bezogen auf die aktuellen Einstellungen. Beachten Sie, dass der "Vorschau"-Button gesperrt ist, wenn "Auto-Vorschau" aktiviert ist (was normalerweise der Fall ist). Die Auto-Vorschau gewährleistet, dass ein Bild sofort aktualisiert wird, wenn sich eine Einstellung verändert. Manchmal kann es hilfreich sein, die Auto-Vorschau auszuschalten (zum Beispiel bei langsamen Systemen). Um dies zu tun, können Sie die "Auto-Vorschau"-Option aus dem Menü "Ansicht" abwählen. Sie können dann den "Vorschau"-Button benutzen, wenn Ihnen PhotoZoom Pro 5 eine auf Ihren Änderungen basierende Vorschau zeigen soll.

Einstellungen
Öffnet das Einstellungen-Fenster, in welchem Sie verschiedene Einstellungen festlegen können.

Hilfe
Öffnet die Benutzeranleitung.

Info
Zeigt Informationen über die Software und ihre Urheber, Versionsnummer, Registrierungsinformationen, Webseiten-Links und zum Copyright an.

Links
Bietet eine einfache Art und Weise, unsere Webseite (www.benvista.com) für aktuelle Informationen über PhotoZoom Pro 5 und andere von uns entwickelte Software zu besuchen. Hier können Sie auch andere BenVista-Programme starten (wenn diese installiert sind) oder direkte Links zu deren Produktinformationsseiten finden.

Das Menü

Unten finden Sie eine Übersicht über alle Menüelemente, einschließlich Tastaturkürzel (sowohl für Windows als auch Mac), falls vorhanden.

Datei
    Öffnen (Strg+O / Alt+O) Öffnet ein Bild
  Speichern als... (Strg+S / Alt+S) Lässt Sie einen Namen, einen Ort und ein Format für das größenveränderte Bild auswählen und speichert dieses
  Öffnen Sie das Beispielbild Öffnet das PhotoZoom Pro 5 Beispielbild
  Zuletzt verwendete Bilder > Öffnet schnell eine kürzlich geöffnete Datei
  Zuletzt verwendete Stapelverarbeitung > Öffnet schnell eine kürzlich gespeicherte Stapelverarbeitung
  Beenden (Nur Windows) Beendet das Programm

Bearbeiten
 Rückgängig (Strg+Z / Alt+Z) Macht die letzte Änderung rückgängig
 Wiederholen (Strg+Y / Alt+Shift+Z) Wiederholt die letzte Änderung
 Auswählen Aktiviert das Auswahlwerkzeug
 Beschneiden Schneidet die aktuelle Auswahl aus (diese Option ist nur aktiv, wenn bereits etwas ausgewählt wurde)
 Abwählen (Strg+D / Alt+D) Entfernt die aktuelle Auswahl (diese Option ist nur aktiv, wenn bereits etwas ausgewählt wurde)
 Horizontal spiegeln Spiegelt das Bild horizontal (links wird zu rechts)
 Vertikal spiegeln Spiegelt das Bild vertikal (oben wird zu unten)
 Nach links drehen Dreht das Bild um 90 Grad gegen den Uhrzeigersinn
 Nach rechts drehen Dreht das Bild um 90 Grad im Uhrzeigersinn
 Bild zurücksetzen Verwirft alle Änderungen, die am Bild gemacht wurden, seitdem es geöffnet wurde (dies kann nicht mehr rückgängig gemacht werden)

Stapelverarbeitung (siehe auch Stapelverarbeitung)
 Neu Öffnet eine neue, leere Stapelverarbeitung
 Öffnen Öffnet eine zuletzt gespeicherte Stapelverarbeitung
 Speichern Speichert die aktuelle Stapelverarbeitung
 Speichern unter... Speichert die aktuelle Stapelverarbeitung unter einem anderen Namen
 Schließen Schließt die aktuelle Stapelverarbeitung
 Bilder hinzufügen... Fügt ein oder mehrere Bilder zur Stapelverarbeitung hinzu
 Ordner hinzufügen... Fügt der Stapelverarbeitung einen Ordner hinzu
 Bild zur Stapelverarbeitung hinzufügen Fügt aktuelles Bild zur Stapelverarbeitung hinzu
 Alle auswählen (Strg+A / Alt+A) Wählt alle Bilder aus, die sich in der Stapelverarbeitung befinden
 Keine auswählen Wählt alle Bilder ab, die sich in der Stapelverarbeitung befinden
 Auswahl entfernen Entfernt die ausgewählten Bilder vom Stapel
 Ausführen (F5) Startet die Stapelverarbeitung mit den gegebenen Einstellungen

Optionen
  Sprache >
 Wählt eine Sprache für PhotoZoom Pro 5 aus
 Einstellungen... Öffnet das Einstellungsfenster

Ansicht

 Auto-Vorschau (an oder aus) Wenn aktiviert, wird das Vorschaufenster nach jeder Veränderung, die Sie machen, aktualisiert
Falls deaktiviert, werden Veränderungen nur gezeigt, nachdem Sie auf den "Vorschau"-Button oder die entsprechende Menü-Option geklickt haben.
 Symbolleiste > Wählt die Größe der Symbolleiste (Aus, Klein, Mittel, Groß)
