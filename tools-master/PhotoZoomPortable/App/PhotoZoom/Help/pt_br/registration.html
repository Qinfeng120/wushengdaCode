Registro

Caso tenha adquirido o PhotoZoom Pro 5, poderá desbloqueá-lo, inserindo o código de registro fornecido com o software. "Unlock code" =======> "Código de registro"

Caso ainda não tenha desbloqueado o software, a janela de desbloqueio será exibida automaticamente, no início do programa (seja na versão dedicada ou em um dos plug-ins):



Algumas senhas ou códigos de registro somente funcionam quando combinados a um determinado nome e endereço de e-mail. Nesses casos, o nome e o endereço de e-mail necessários serão claramente especificados com o código de registro. Certifique-se de preencher todas as informações exatamente da forma especificada.

If you have purchased this software as an upgrade, the software may ask you to provide additional unlock info. In such case, fill in all info exactly as indicated by the software.

A sua senha (código de registro) somente será necessária uma vez para desbloquear o software em caráter permanente. Após a instalação das atualizações grátis, não haverá mais necessidade de inseri-la novamente.

Observe que enquanto não tiver sido desbloqueado, o software estará sendo executado no modo de teste, com a seguinte restrição: as imagens que o programa salvar exibirão uma marca d'água. A janela de desbloqueio pode ser aberta enquanto o software está sendo executado no modo teste, selecionando-se "Unlock..." (Desbloquear) a partir do menu de Ajuda.

Caso tenha adquirido o seu software através da Internet, e tenha recebido seu código de registro via e-mail, fazemos forte recomendação para que preencha os campos do código de registro usando copiar/colar para evitar erros de digitação.
