Traitement par lot

Cette fonction permet de redimensionner simultanément plusieurs images. Vous pouvez spécifier les mêmes paramètres pour toutes les images ou utiliser différentes valeurs pour différents groupes d'images. Choisissez ce que vous préférez et PhotoZoom Pro 2 fera le reste !

Lorsque vous cliquez sur le bouton "Nouveau lot", la fenêtre de création de lot est affichée au bas de l'écran :

1) Sélectionnez les images à traiter
Le bouton "Ajouter des images..." vous permet d'ajouter une ou plusieurs images au lot en cours de création. Pour ajouter la totalité d'un dossier, utilisez le bouton "Ajouter un dossier...". En cochant la case "Inclure les sous-dossiers" avant de sélectionner un dossier, vous autorisez l'addition de toutes les images situées dans tous les sous-dossiers.

2) Liste des images
C'est l'endroit où toutes les images constituant le lot sont listées avec les informations nécessaires : taille d'origine, informations sur le redimensionnement, taille de sortie. Si vous changez les paramètres (par exemple la méthode d'interpolation), l'information sera mise à jour dans cette liste. Les modifications ne s'appliqueront qu'aux images sélectionnées dans cette liste. Vous pouvez donc ne sélectionner qu'une seule image, un groupe d'images ou toutes les images.

Les modifications de sélection peuvent se faire de plusieurs façons :

  • Pour ajouter une image à la sélection, maintenez la touche Ctrl (Windows) ou Cmd (Mac) enfoncée, puis cliquez sur les images que vous souhaitez inclure dans la sélection

  • Pour sélectionner un groupe d'images, cliquez sur une première image. En maintenant la touche "Maj" enfoncée, cliquez sur une autre image. Les deux images, ainsi que toutes celles situées entre elles, seront sélectionnées

3) Boutons de sélection
Tout sélectionner : Sélectionne toutes les images de la liste
Ne rien sélectionner : désélectionne toutes les images déjà sélectionnées
Suppression de la sélection : Supprime toutes les images sélectionnées de la liste

4) Paramètres de redimensionnement
C'est ici que vous spécifierez les nouvelles dimensions et les paramètres de la méthode d'interpolation pour les images définies dans le lot. Pour toute information sur ces paramètres, consultez le paragraphe Le logiciel expliqué (paragraphes 2 & 3) et Réglages avancés. Ces changements ne seront appliqués qu'aux fichiers sélectionnés.

Attention : si plusieurs images sont sélectionnées, certaines valeurs peuvent ne pas apparaître dans la partie "Nouvelles dimensions". Cela dépend de la taille des images sélectionnées et de la manière dont les paramètres de redimensionnement sont spécifiés (par exemple pixels et pourcentage créent une différence). Il peut en résulter qu'une valeur ne puisse être affichée. Dans ce cas, PhotoZoom Pro 2 laissera la valeur à blanc. Cependant, vous pourrez toujours trouver les informations relatives à chaque image dans la liste du lot.

5) Exécuter
Lorsque tous les paramètres ont été sélectionnés, cliquez sur le bouton "Exécuter". La fenêtre suivante est alors affichée :



Si vous souhaitez changer le dossier de sauvegarde, cliquez sur le bouton "Parcourir...". Si vous indiquez un dossier qui n'existe pas, PhotoZoom Pro 5 vous demandera si vous souhaitez qu'il le crée pour vous.

Avec l'option "Format de sortie", vous pouvez choisir de sauvegarder vos images en format TIFF, JPEG, PNG, BMP ou TGA. Le format TIFF est souvent utilisé car il supporte un nombre important de fonctionnalités (consultez Formats d'image. Pour des tailles de fichiers minimales, utilisez le format JPEG.

Pour terminer, vous pouvez décider ce que vous souhaitez que fasse PhotoZoom Pro 5 en cas de sauvegarde de fichier, si un fichier du même nom existe déjà dans le dossier.

Prêt à démarrer ? Cliquez sur "Exécuter" .



La fenêtre d'avancement du traitement fournit une estimation sur le temps nécessaire au traitement complet de l'image par PhotoZoom Pro 5. Cette information est continuellement mise à jour. Ce temps dépend du nombre d'images à traiter et des options choisies. Il faut également noter que le traitement par lot nécessite une puissance de calcul importante, ce qui peut dégrader les performances des autres applications s'exécutant simultanément sur votre ordinateur. Vous pouvez provisoirement suspendre l'exécution de PhotoZoom Pro 2 et la reprendre à un moment plus approprié.
